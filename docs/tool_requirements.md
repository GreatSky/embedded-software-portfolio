# Tool Requirements

- **GCC 9+**: C/C++ compilation.
- **Python 3.8+**: Automation.
- **GDB/Valgrind**: Debugging.
- **JTAG Debugger**: Firmware flashing.
- **CUnit**: Unit testing.
- **Yocto/Buildroot**: Linux builds.
- **Vector CANoe**: I2C/SPI/CAN testing.
- **Wireshark**: TCP/IP analysis.
- **Jira**: Issue tracking.
